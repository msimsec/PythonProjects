birinci bölümde, 
instagram.py dosyasında instagram web adresi çağrılmakta, instagramUserInfo.py sayfasından gelen kullanıcı adı ve password bilgileri otomatik çekilerek, 
login olma işlemi gerçekleşmektedir. 
ikinci bölümde,

üçüncü bölümde,
